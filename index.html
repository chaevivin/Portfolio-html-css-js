네캔만원은 편의점 맥주의 다양한 정보를 제공하고 이를 기반으로 한 커뮤니티를 형성하여 맥주 구매 결정에 도움을 주는 서비스입니다. 전국의 편의점은 5만개 이상 존재하고 맥주는 편의점의 주력 판매 물품 중 하나지만, 많은 맥주 종류에 대한 정보의 부재로 구매에 어려움을 겪는 소비자를 위해 기획되었습니다.

기존의 맥주 추천 서비스들은 맥주의 단순한 정보 (맥주 이름, 도수 등)만 제공해주거나, 정보 제공보다 광고 위주인 곳이 대다수입니다. 네캔만원은 기존 서비스와 차별화를 두기 위해 편의점에서 판매하는 맥주의 다양한 정보 (맥주 종류, 도수, 향, 맛, 설명 등)를 제공하고 많은 사람들과 의견을 나눌 수 있는 리뷰 기능을 추가하여 소통의 장을 마련하였습니다. 그리고 맥주를 종류 (에일, 라거, IPA, 스타우트) 별로 필터링해서 검색할 수 있고, ‘찜하기’ 버튼을 제공하여 사용자가 찜한 맥주 리스트를 관리할 수 있습니다.

담당 업무

1. 맥주를 종류별로 확인할 수 있는 다중 선택 필터 기능

‘라거’, ‘에일’, ‘IPA’, ‘스타우트’ 4개의 맥주 종류 버튼을 클릭할 때마다 해당하는 맥주 리스트들을 보여주는 기능을 구현하였습니다. 다중선택이 가능하도록 useCallback을 이용하여 배열에 현재 클릭된 필터를 모아주었습니다. 클릭된 필터는 배열에 추가되고, 이 배열은 axios로 백엔드에 해당 필터의 맥주 리스트를 요청하고 리스트를 받아오면, 리액트가 컴포넌트를 리렌더링해서 사용자가 필터를 클릭한 순간 바로 리스트를 볼 수 있도록 구현하였습니다. 블로그에서 더 자세한 내용을 알아보세요.

2. 사용자 경험을 바탕으로 한 별점 기능

UX를 가장 많이 신경 쓴 기능 중 하나입니다. 별점 기능에 대한 API가 많이 존재했지만 한 번 클릭한 별을 다시 취소할 수 없다거나, 별점이 소수점 단위까지 기록되어 별을 클릭하면 클릭한 위치까지만 별이 채워지는 등 사용자가 쉽게 사용할 수 없는 부분들이 존재하였습니다. 네캔만원은 기존의 별점 API보다 사용자가 더 쉽고 편리하게 접근할 수 있도록 구현하였습니다. 블로그에서 더 자세한 내용을 알아보세요.

3. 실시간 댓글 기능

맥주에 대한 리뷰를 등록하면 ‘리뷰’ 페이지에 실시간으로 리뷰를 볼 수 있도록 하였습니다. 리뷰는 유튜브 댓글 형식 처럼 구현하였고 다른 사용자들이 대댓글을 달 수 있도록 하였습니다. 리뷰 (댓글) 오른쪽 하단에 있는 토글 버튼을 누르면 해당 리뷰 (댓글)의 대댓글을 볼 수 있고, 대댓글을 달 수 있는 입력창을 구현하였습니다. 실시간 댓글 기능은 네캔만원의 핵심 기능(목표)으로 사용자들이 맥주에 대해 평가하고 실시간으로 의견을 공유할 수 있으며, 단순한 설명으로는 얻을 수 없는 정보를 바탕으로 기존 구매자와 사용자가 공감대를 형성하여 새로운 맥주를 시도해볼 수 있는 경험으로 이어질 수 있습니다.

놀러와요! 모현의 숲은 한국외대 글로벌캠퍼스에 다양한 동물들이 출연하는 특색을 살린 동물 테스트로 글캠생들의 정체성 확립과 캠퍼스 홍보 효과를 위해 기획되었습니다. "나라", "직업", "동물"의 3가지 요소로 새로운 동물 캐릭터를 생성합니다. "나라"는 글로벌캠퍼스의 다양성을, "직업"은 학과의 종류를, "동물"은 캠퍼스의 자연환경을 상징합니다. 테스트 결과는 "(나라)에서 온 (직업)하는 (동물)" 형식으로 표기됩니다. 그리고 사용자를 확보하기 위해 각종 SNS 공유 기능을 추가하고, 최종적으로 CTA를 이용하여 사용자를 메타버스 플랫폼으로 유도하였습니다.

담당 업무

1. 전반적인 화면 구현

프론트엔드를 맡아 프로젝트의 모든 페이지를 구현하였습니다. 사용자가 스마트폰으로 쉽게 테스트할 수 있도록 모바일 퍼스트(Mobile First), 반응형 웹으로 구현하였습니다.

2. 아이콘과 함께 움직이는 진행바 (프로그레스바)

사용자에게 테스트의 진행 상황을 보여주기 위해 상단에 진행바를 구현하였습니다. 아이콘과 함께 움직일 수 있도록 CSS의 calc() 함수를 사용하여 페이지가 넘어갈때마다 아이콘의 위치를 변경해 주었습니다. 블로그에서 더 자세한 내용을 알아보세요.

3. SNS 공유 기능

서비스의 홍보 효과를 높이기 위해 접근성이 좋은 sns에 공유할 수 있는 기능을 추가하였습니다. 트위터, 페이스북, 카카오톡에 공유할 수 있으며, 로고가 그려진 버튼을 클릭하면 해당 sns로 테스트가 공유됩니다. 카카오톡 공유 기능은 JavaScript SDK를 이용한 카카오톡 공유 API를 사용하여 개발하였습니다.

4. 다양한 분야의 사람들과의 협업

프로젝트에 기획자와 디자이너가 참여하여 직접적으로 소통하며 프로젝트를 완성해 나갔습니다. 기획자와 Slack이나 Notion으로 기획의도와 맞게 테스트 내용을 함께 구성하고, 스크립트를 작성하였습니다. 디자이너와 Figma를 사용하여 UX에 대한 고민을 기반으로 페이지의 레이아웃을 함께 구성하였습니다.
